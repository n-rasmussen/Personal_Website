Game using PyGame

Using PyGame I made a simple car game to avoid traffic cones. This game involves event handling, moving sprites, scoring, and a start menu. The score is how long your car avoids hitting a traffic cone. The frequency and speed of the traffic cones increase with time. Not all cones more at the same speed. The highest score is kept between games in the same runtime.
